# GSD Hello World — 網頁計算機

一個最簡單的網頁計算機，純學習用途。單一 `index.html`，用原生 HTML + JavaScript 實作，零框架、零建置工具、零後端。

## 功能

- 兩個數字輸入框
- 「加」「減」「乘」「除」四顆運算按鈕
- 一個結果顯示區塊
- 除以零、空白或非數字輸入時，顯示可理解的中文錯誤提示，頁面不會壞掉

## 怎麼用

直接用瀏覽器開啟 `index.html` 即可（`file://` 協定，不需伺服器、不需安裝任何東西）：

```
# 直接雙擊檔案，或在瀏覽器網址列開啟
file:///path/to/index.html
```

輸入兩個數字 → 按下任一運算按鈕 → 結果區塊顯示答案。

## 技術說明

- 純前端單檔，無依賴
- 結果以 `textContent` 輸出（不用 `innerHTML`），運算使用原生算術運算子（不用 `eval()`）
- 範圍刻意維持最小：只做基本四則，不含括號、百分比、科學函式或連續運算
